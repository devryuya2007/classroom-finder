1. サポート概要

Classroom-Finder は、Google Classroom 上のコースとストリーム投稿を高速に検索するための Chrome 拡張機能です。本ページでは、基本的な操作方法やトラブルシューティング、問い合わせ先を案内します。Google LLC とは提携していません。

2. よくある質問 (FAQ)

  • 拡張をインストールするには？ Chrome ウェブストアから「Classroom-Finder」を追加するだけで利用できます。開発版を試す場合のみ、chrome://extensions で「デベロッパーモード」を有効にしてフォルダを読み込んでください。
  • 検索バーが表示されません。 Classroom ページを再読み込みし、拡張が有効になっていることを確認してください。続くセクションの手順で詳細を確認できます。
  • Google アカウントの許可を取り消したい。 Google アカウント設定 > セキュリティ > アカウントへのアクセスを許可したアプリから、本拡張を選択してアクセス権を削除できます。

3. トラブルシューティング

  1. Classroom ページを再読み込みし、Chrome アドレスバー右の拡張アイコン表示を確認する。
  2. chrome://extensions を開き、「更新」ボタンで拡張を再読み込みする。
  3. 「エラーを表示」をクリックし、サービスワーカーやコンテンツスクリプトのログを確認する。
  4. 必要に応じて拡張をアンインストールし、再インストールする。

問題が解決しない場合は、ログ情報（エラー内容、再現手順など）を添えてお問い合わせください。

4. データとプライバシー

本拡張が扱うデータや保護方針については、プライバシーポリシー を参照してください。取得した Classroom データはすべて端末内に保存され、開発者サーバーには送信されません。

1. Support Overview

Classroom-Finder is a Chrome extension that helps you search Google Classroom courses and stream posts quickly. This page summarizes usage tips, troubleshooting steps, and contact information. The project is independently maintained and not affiliated with Google LLC.

2. Frequently Asked Questions (FAQ)

  • How do I install the extension? Install “Classroom-Finder” directly from the Chrome Web Store. Loading the folder via chrome://extensions is only needed when testing the development build.
  • The search bar does not appear. Reload the Classroom page and ensure the extension is enabled. Follow the troubleshooting steps below if the issue persists.
  • How can I revoke Google account access? Open Google Account settings > Security > Third-party apps with account access, select this extension, and remove access.

3. Troubleshooting

  1. Reload the Classroom page and check whether the extension icon appears next to the Chrome address bar.
  2. Open chrome://extensions and click “Reload” for the extension.
  3. Click “View errors” to review service worker or content script logs.
  4. Uninstall and reinstall the extension if necessary.

If the problem continues, please contact us with logs, error messages, and reproduction steps.

4. Data & Privacy

For details about data handling, please review the Privacy Policy. All Classroom data stays on your device and is never sent to developer-controlled servers.
