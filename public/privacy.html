Classroom-Finder
プライバシーポリシー

最終更新日 / Last Updated: 2025-10-07

Classroom-Finder は Google Classroom の投稿・コースを高速検索するための Chrome 拡張機能です。本ポリシーでは、取得するデータ、利用目的、ユーザーが管理できる選択肢を明確に示します。

取得範囲

Classroom API の読み取りスコープのみ（コース／お知らせ）。

保存場所

端末のブラウザ内（IndexedDB / chrome.storage）。開発者サーバー送信なし。

利用目的

投稿検索とナビゲーション支援のみ。広告・販売・第三者提供は不実施。

1. このポリシーについて

本プライバシーポリシーは、Chrome 拡張機能「Classroom-Finder」（以下「本拡張」）が Google Classroom のデータをどのように取り扱い、ユーザーのプライバシーを保護するかを説明するものです。本拡張は個人で運営され、Google LLC またはその関連会社とは提携していません。

2. 収集・利用する情報

  • Google Classroom API を通じて取得するコース情報（コース名、ID、担当教員名など）
  • 各コースのストリーム投稿・お知らせ（本文、タイムスタンプ、添付ファイルのメタデータ）
  • 本拡張が内部で生成する検索用インデックスおよびユーザー設定（ブラウザの IndexedDB および chrome.storage に保存）

これらの情報は、Chrome の chrome.identity API を利用し、ユーザーが明示的に許可した OAuth スコープ（https://www.googleapis.com/auth/classroom.courses.readonly および https://www.googleapis.com/auth/classroom.announcements.readonly）の範囲内で取得します。

3. 利用目的

取得した情報は、Google Classroom 上の投稿やコースを高速に検索・ナビゲーションするためだけに使用します。広告配信、トラッキング、プロファイリング、第三者への販売・共有には一切利用しません。

4. 法的根拠

EU/EEA 居住者に対しては、本人の同意または正当な利益に基づいて処理します。必要に応じて同意はいつでも取り消すことができ、処理の停止を求める権利があります。

5. 保管場所と期間

すべてのデータはユーザーのブラウザ内（IndexedDB 等）にのみ保存され、開発者サーバーや第三者のサーバーには送信されません。データはユーザーが削除するまで保持されます。以下の方法で削除できます。

  • Chrome の設定から本拡張をアンインストールする
  • Chrome の拡張データをクリアする（chrome://settings/siteData 等）
  • Google アカウントの「アカウントへのアクセスを許可したアプリ」から本拡張のアクセス権を取り消す

6. 第三者提供および国際移転

本拡張は取得した Classroom データを第三者（開発者を含む）と共有せず、クラウドサーバーにも送信しません。よって国際的なデータ移転も発生しません。

7. 安全管理措置

  • HTTPS 通信のみを利用し、classroom.googleapis.com への読み取り専用 API コールに限定
  • Chrome Web Store マニフェストで不要なホスト権限・メソッドを付与しない設計
  • ブラウザ内保存データは Chrome のサンドボックス機構により保護され、開発者は直接アクセスできません

8. ユーザーの権利

ユーザーはいつでもデータの閲覧・削除・同期停止が可能です。EU/EEA 居住者は GDPR に基づくアクセス権・訂正権・削除権・処理制限権・異議申立権を有し、米カリフォルニア州居住者は CCPA/CPRA に基づく開示・削除・オプトアウトの権利を有します。権利行使や問い合わせは下記連絡先までご連絡ください。

9. 児童・生徒のプライバシー

本拡張は教育機関での利用を想定していますが、児童生徒の個人情報を開発者が収集・保管・再配布することはありません。教育機関や保護者は、自組織の方針に基づき利用の可否をご判断ください。

10. Google API Services ポリシーの遵守

本拡張は Google API Services User Data Policy（特に Limited Use 要件）に準拠し、受領したデータを本拡張の機能提供以外の目的で使用しません。

11. 変更について

1. About This Policy

This privacy policy explains how the Chrome extension “Classroom-Finder” (the “Extension”) processes Google Classroom data and protects user privacy. The Extension is operated by an independent developer and is not affiliated with Google LLC or its affiliates.

2. Data We Collect and Use

  • Course metadata retrieved via the Google Classroom API (course name, ID, instructor name, etc.)
  • Stream announcements for enrolled courses (content, timestamps, attachment metadata)
  • Local search indexes and preference data stored in the browser (IndexedDB and chrome.storage)

All data is obtained through Chrome’s chrome.identity API using the OAuth scopes expressly granted by the user (https://www.googleapis.com/auth/classroom.courses.readonly and https://www.googleapis.com/auth/classroom.announcements.readonly).

3. Purpose of Processing

The collected information is used solely to help users search and navigate their Google Classroom content more efficiently. It is never used for advertising, tracking, profiling, or sold/shared with third parties.

4. Legal Bases

For users located in the EU/EEA, processing is based on user consent or the Extension’s legitimate interests. Consent can be withdrawn at any time, and you may request restriction or termination of processing.

5. Storage Location and Retention

All data remains on the user’s device (IndexedDB, chrome.storage). No data is transmitted to the developer’s servers or any third party. Users can delete the data at any time by uninstalling the extension, clearing extension storage, or revoking the extension’s access in their Google Account settings.

6. Sharing and International Transfers

The Extension does not share Classroom data with any third party and does not send it to cloud services; therefore, no international data transfer occurs.

7. Security Measures

  • API calls are restricted to HTTPS requests against classroom.googleapis.com and limited to read-only HTTP methods.
  • The manifest is configured with the minimum required permissions and host access.
  • Stored data is handled by Chrome’s sandboxed storage; the developer cannot access it directly.

8. Your Rights

Users may view, delete, or stop syncing their data at any time. Residents of the EU/EEA have rights under the GDPR (access, rectification, erasure, restriction, objection). California residents have rights under the CCPA/CPRA (access, deletion, opt-out). To exercise these rights, contact the email address listed below.

9. Children and Students

The Extension is intended to assist educators and students within Google Classroom. The developer does not collect, store, or redistribute student information. Schools and guardians should ensure usage complies with their policies.

10. Compliance with Google API Services Policy

The Extension complies with the Google API Services User Data Policy, including the Limited Use requirements. Data received from Google APIs is used only to provide the Extension’s functionality.
